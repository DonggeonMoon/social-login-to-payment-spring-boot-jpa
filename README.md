# SocialLoginToPayment-Spring_Boot-JPA
소셜 로그인, 결제 페이지 만들기

구현할 기능
* 소셜 로그인
  * 구글
  * 페이스북
  * 트위터
  * 네이버
  * 카카오
  * 깃허브(미정)

* 결제 모듈 사용
  * 이니시스
  * 토스페이먼츠
  * KCP
  * 아임포트(미정)
